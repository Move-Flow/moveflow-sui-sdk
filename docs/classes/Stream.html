createTransaction

  • createTransaction(coinType, name, remark, sender, recipient, depositAmount, startTime, stopTime, interval?, closeable?, modifiable?): Promise<TransactionBlock>

  • This function builds a TransactionBlock to create a new payment stream

    Parameters

    • coinType: string

      the coin type that is used in the payment stream, e.g., 0x2::sui::SUI

    • name: string

      the name of the stream, max 1024 characters

    • remark: string

      the remark of the stream, max 1024 characters

    • sender: string

      the payment sender's address, which must be the same as the TransactionBlock signer

    • recipient: string

      the payment receiver's address

    • depositAmount: bigint

      the initial deposit amount of the coin specified by the coinType parameter

    • startTime: number

      the unix epoch time in seconds, i.e., Date.now() / 1000

    • stopTime: number

      the unix epoch time in seconds

    • interval: number = 60

      the payment interval in seconds, and the deposit is divided evenly among the intervals. Default to 60 seconds

    • closeable: boolean = true

      if the stream can be closed by the sender. Default to true

    • modifiable: boolean = true

      if the stream can be modified by the recipient. Default to true. Not effective now

    Returns Promise<TransactionBlock>

    the TransactionBlock which can be signed to create a new payment stream
